# MinVideo 

MinVideo 是一个 AI 驱动的视频生成平台，旨在帮助用户利用先进的 AI 模型创
建高质量的产品视频、克隆视频风格，并将静态图像转化为引人入胜的动态视频内
容。

## 🚀 核心功能

- **克隆视频 (Clone Video)**：分析输入视频以提取其风格、节奏和吸引点 
(Hook)，然后生成具有相似特征的新视频。
- **图生视频 (Image to Video)**：瞬间将静态产品图片转化为动态、吸睛的
视频。

我想分享一下我最近正在开发的 Side Project —— MinVideo。

做过电商或者内容运营的朋友都知道，现在是短视频的时代。但是制作高质量的视频门槛太高了，外包又贵。为了解决这个问题，我开发了这个 AI 视频生成平台。

它主要解决两个问题：

不知道怎么剪出爆款感？用「克隆视频 (Clone Video)」 你只需要上传一个你觉得很棒的参考视频（比如竞品的爆款），AI 会自动提取它的风格、节奏和吸睛点 (Hook)，然后生成一个具有同样“爆款基因”但内容完全属于你的新视频。

只有产品白底图？用「图生视频 (Image to Video)」 瞬间把静态的产品图片转化为动态视频。对于做 Dropshipping 或者亚马逊的朋友来说，这能极大提高转化率。
